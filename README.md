## Dependencies

| Depencency | Version |
| ---------- | ------- |
| lottie-ios | 3.1.5   |
| SwiftLint  | 0.38.1  |
| SnapKit    | 5.0.0   |

## Installation

```shell

$ git clone https://github.com/Nexters/Yetda_iOS.git
$ cd Yetda_iOS
$ pod install
```
